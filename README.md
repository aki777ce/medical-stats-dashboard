# 救急受け入れ停止データ分析ダッシュボード

このアプリケーションは、医療機関の救急受け入れ停止データを分析し、インタラクティブに統計情報を表示するためのウェブベースのダッシュボードです。

## 主な機能

- Excelデータのアップロードと解析
- 平日日動帯（08:00-17:00、土日祝日を除く）のストップ時間分析
- 年別・月別の統計表示（全データ/平日日動帯の切り替え表示可能）
- 診療科別（特に循環器科）のデータ分析
- ストップ理由別の統計
- インタラクティブなフィルタリング
- CSVエクスポート機能

## 使用方法

1. `index.html`をウェブブラウザで開きます
2. 「ファイルを選択」ボタンをクリックし、Excelファイルをアップロードします
3. アップロードされたデータに基づいてダッシュボードが自動的に更新されます
4. フィルターを使用して特定の期間や診療科のデータを表示できます

## データ形式

アップロードするExcelファイルには以下の列が必要です：

- `日付`: ストップが発生した日付（YYYY/MM/DD形式）
- `ストップ科`: ストップが発生した診療科
- `停止時間`: ストップの時間範囲（HH:MM～HH:MM形式）
- `理由`: ストップの理由

## 特別な分析

- **平日日勤帯分析**: 平日（土日祝日を除く）の8:00から17:00までの時間帯を「平日日勤帯」として定義し、この時間帯に含まれるストップ時間を特別に分析します
- **循環器科データ**: 循環器科のデータを特に注目して分析できます
- **理由別分析**: ストップ理由ごとの統計を確認できます

## チーム共有

このアプリケーションはHTMLベースで作成されているため、以下の方法で特定メンバーと共有できます：

1. ファイル共有サービス（OneDrive、Google Driveなど）を使用
2. 社内イントラネットにアップロード
3. 簡易的なウェブサーバーでホスティング

## 技術仕様

- HTML5, CSS3, JavaScript (ES6+)
- Chart.js 3.9.1: グラフ描画
- SheetJS (xlsx): Excelファイル解析
- Moment.js: 日付・時間処理
- Bootstrap 5: UIフレームワーク
- Japanese Holidays: 日本の祝日判定

## 注意事項

- ブラウザ上で動作するため、アップロードされたデータはサーバーに送信されません
- 大量のデータを処理する場合は、ブラウザのパフォーマンスに影響する可能性があります
